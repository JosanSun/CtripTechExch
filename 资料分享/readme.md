### 如何参加分享？

首先，要有满足**分享原则**【见下】的资料。然后，联系@[Josan](https://github.com/JosanSun)，分享资料，由Josan更新主分享链接。

### 分享原则

1. 高清版pdf

2. 有书签

3. 如有代码，必须含有配套代码  【注意版本的对应】; 同时，代码的分享必须是压缩包的形式。  
- 【为什么压缩？ 百度云一次保存超过500个文件，就会报错。压缩可以一定程度解决这个问题。】

4. 豆瓣评分至少7分以上。
